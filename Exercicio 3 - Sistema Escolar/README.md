## Installation

1. Clone the repository:

```bash
git clone https://github.com/williambreda/sistemas-distribuidos
```

2. Install dependencies with Maven

3. Install [PostgresSQL](https://www.postgresql.org/)

## Usage

1. Start the application with Maven
2. The API will be accessible at http://localhost:8080


## API Endpoints
Endpoints da aplicação podem ser encontrados abaixo:

```markdown
http://localhost:8080/swagger-ui/index.html#/
```

## Authentication
As API's utilizam Spring Security para autenticação e roles.

```
USER -> Usuário padrão, acesso restrito nas API's
ADMIN -> Usuário com acesso a todas API's
```
To access protected endpoints as an ADMIN user, provide the appropriate authentication credentials in the request header.

## Database
The project utilizes [PostgresSQL](https://www.postgresql.org/) as the database. The necessary database migrations are managed using Flyway.
